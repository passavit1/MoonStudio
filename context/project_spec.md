# Project Overview

## 1. Goal
A web-based sales dashboard for MoonStudio to track 3D printing model sales across TikTok Shop, Shopee, and Lazada.

## 2. Core Users
- MoonStudio owner (business management and sales analysis).

## 3. Key Success Metrics
- Seamless CSV import for all three platforms.
- Accurate revenue and order volume tracking.
- Product performance insights (e.g., top variations).
- Multi-platform sales comparison.

## 4. Platform Data Sources
- **TikTok Shop:** Monthly CSV exports (Sample: `data/2026-03.csv`).
- **Shopee:** Monthly CSV exports (Pending sample).
- **Lazada:** Monthly CSV exports (Pending sample).

## 5. Primary Features (V1)
- **Dashboard UI:** Clean, responsive, and visual (charts).
- **CSV Uploader:** A tool to process and store platform-specific data.
- **Sales Analytics:** Total Sales, Average Order Value (AOV), Order Status Breakdown.
- **Product Analytics:** Units sold per SKU/Variation.
